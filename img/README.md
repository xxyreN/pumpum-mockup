# How images work in this mockup

## The system is automatic.

1. **Open the site** (double-click `index.html` in the parent folder)
2. **Every missing image displays its exact filename** where it should go — you'll see it as white text on a dark tile
3. **Save the IG screenshot with that exact name** into this folder
4. **Refresh the browser** (Ctrl+Shift+R for hard reload) — the image appears automatically

No configuration. No code edits. Just match filename → save → refresh.

## How to save from Instagram

1. Go to the IG post on desktop
2. Right-click the image → "Save image as…"
3. Name it exactly as above (e.g. `hero-yellow-wall.png`)
4. Save to: `E:\claude projects\webagency-madrid\websiteagency\pumpum-mockup\img\`
5. Refresh the page (Ctrl+Shift+R)

If right-click doesn't work, screenshot the image instead (Win+Shift+S), paste into Paint, save as .png with the correct name.

## Fallback behaviour

Any image you skip → shows its filename on a dark tile. Nothing breaks. The site always renders.
